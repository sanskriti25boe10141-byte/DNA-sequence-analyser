 # DNA Sequence Analyzer

# Description
This project is a Python-based tool developed to analyze DNA sequences. It helps in understanding basic bioinformatics concepts by performing simple analysis on DNA data.

# Objective
The objective of this project is to create a simple tool that can analyze DNA sequences and help understand basic concepts of bioinformatics using Python.

# Features
- Validates DNA sequence (A, T, G, C only)
- Counts number of nucleotides
- Calculates GC content
- Generates reverse complement of DNA sequence

# Technologies Used
- Python (basic concepts like functions, loops, and conditions)

# How to Run
1. Install Python on your system
2. Download the file `dna_analyzer.py`
3. Run the program using:
   python dna_analyzer.py

# Example Input
ATGCGTA

# Example Output
Valid DNA sequence
Nucleotide Count: A=2, T=2, G=2, C=1
GC Content: 42.86%
Reverse Complement: TACGCAT
